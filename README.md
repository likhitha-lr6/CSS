# CSS
CSS (Cascading Style Sheets) is used to style and design web pages. It controls colors, fonts, spacing, and layout of HTML elements. CSS helps make websites look attractive and organized. It also allows web pages to adjust properly on different devices like mobiles, tablets, and computers.
In short, language to describe the style of a document
Ex-
h1{
    color:red;
 }
 Here h1= selector; color= property; red= value
 Selectors:Universal selector {} ; Element selector h1{} ; ID Selector #myId{} ; Class selector .myclass{}
